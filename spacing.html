    Spacing

    Sizes

    All heights and widths are calculated from a base unit.

    4px | base


    8px | xs


    12px | s


    20px | reg


    40px | m


    100px | l


    148px | xl

    Function

    tn-space($size, $modifiable)

    Accepts either keyword or increments for size. The second argument indicates whether the value is modifiable based on the $tn-space-modify feature flag.

    Example:

    .foo {
    height: tn-space(20); //80px
    padding: tn-space(m); //40px
}

    If $tn-space-modify is set to 'compact', the spacing value will be cut in half. Primarily enabled for collection components when information density is greater.

    Example:

    .bar {
    padding: tn-space(m,true); //20px
}
    Acceptable arguments
    $size base,xs,s,reg (default),m,l,xl
    $modifiable true, false (default)
