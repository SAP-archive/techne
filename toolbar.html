    Toolbar

    The Toolbar component is used always along a collection of items, like cards or a table. The main use of it is to manipulate the collection. We provide elements and interaction patters for filtering, sorting, pagination and view changes.

    Filtering

    All the filtering options will be under the filtering icon () located on the right of the toolbar. When pressed it will display a second row on the toolbar, from this second row will show all available filtering options for the user to select.

    The selected filters will be displayed on a new row bellow. Every filter will include an remove icon () to deselect it. Also a “Clear all” button that will deselect all filters.

    Sorting

    The Toolbar sorting elements modify the order of the items in the collection depending on the user selection. It is located right besides the filter icon.
