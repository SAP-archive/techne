1.1.2 - Toggle Buttons

A majority of buttons in the site are built from the same base class.

Default styling

:hover

 
<div data-toggle="buttons" class="btn-group">
	<label class="btn btn-toggle btn-lg active {$modifiers} ">
		<input type="radio" id="option1" name="options"> On
	</label>
	<label class="btn btn-toggle btn-lg {$modifiers} ">
		<input type="radio" id="option2" name="options"> Off
	</label>
</div>
