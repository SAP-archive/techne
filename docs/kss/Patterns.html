Patterns & Layouts

hybris techne allows for an immersive, consumer-like experience for business users. Using the components provided in the component library, achieve patterns and mobile-first page layouts as exemplified below.


List Pages

Lists are a high-utility pattern that allows for quick sorting, filtering and scanning. 'Add a X' ideally is found at the top of the list pattern for users to quickly add data in context.

Card Pages

Cards allow for a highly-visual approach for browsing, scanning and mental information organization. Cards can be used with or without a visual component, or with different patterns of visual components to allow for data visualization and seperation.

Retail Product Pages

Specific for implementation in need of high-touch product information management and administration, the product admin layout provides and immersive, WYSIWYG enviorenment for business users to work within.

Trees

Trees are a great way to display and manage hierarchy in a visual, drag-and-drop manner.

Mixed Pattern Pages

Mix patterns to create page layouts that are highly intuitive and contextual for business users

Tours

A “Tour” should be used when a new visitor comes to a UI module. It gives a step-by-step explanatino of what the UI Module covers and where to go to accomplish certain tasks. The ‘Tour’ pattern uses the “Tour Bubble” UI component.
