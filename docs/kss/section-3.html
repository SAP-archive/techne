Reusable elements that can be used across the site. So far this just includes toggles.

A majority of buttons in the site are built from the same base class. Deprecated - This component will be replaced in the future please don't use it anymore

:hover
<div data-toggle="buttons" class="btn-group">
	<label class="btn btn-toggle btn-lg active {$modifiers} ">
		<input type="radio" id="option1" name="options"> On
	</label>

	</label>
