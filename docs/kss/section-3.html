3 Toggles

Reusable elements that can be used across the site. So far this just includes toggles.

3.1 Small Toggle

Small toggle button that is used for small inline indication of a status in combination with a link 

<span class="y-toggle">
  <input type="checkbox" id="toggleid"/>
  <label for="toggleid">Labeltext (optional)</label>
</span>

3.2 Large Toggle

Large toggle button that is used for small inline indication of a status in combination with a link 

<span class="y-toggle y-toggle-lg">
  <input type="checkbox" id="toggleidlg"/>
  <label for="toggleidlg">Labeltext (optional)</label>
</span>
