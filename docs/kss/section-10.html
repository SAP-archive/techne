10 Forms

Reusable elements that can be used across the site. So far this just includes form elements.

10.1 Custom Checkboxes

Checkboxes adhere to typical use; when multiple selections for an item are available. Check Boxes should be placed in proximity to the value needing selection(s). Deprecated

<div class="checkbox">
 <input type="checkbox" id="optionsCheck1" value="optionc1" checked>
 <label for="optionsCheck1">Check me out</label>
</div>

Read Only Form Elements share the same styling as the well know input-lg, input-sm size modifier classes. Use them if you want to provide inline text with a label within a form.

10.4 Inline Help

Use inline help to guide the user through formular fields. Focus on the action required and make it relevant to the user’s current situation.

Do explain:

  • unfamiliar concepts
  • what format the information should be given in
  • where to find further information

The inline help should be placed below the input field. Keep it short and clear. Avoid more then two lines.

This line is reserved for help text. 
 <form>
   <label>Title</label>
   <input type="text" class="form-control">
   <span class="inline-hint-text">This line is reserved for help text.</span>
 </form>
