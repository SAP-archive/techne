Mixins

@mixin tn-type($size, $font, $weight, $transform)

Each type size has default family, weight and transform rules built in. Passing the size alone will return all default rules.

Example:

.foo {
    @include tn-type(5);
}

Additional arguments can be passed to the mixin.

Example:

.small-header {
    @include tn-type(-3,header,reg,none);
}
Acceptable arguments
$size -3,-2,-1,0 (default),1,2,3,4,5
$font body (default), header, code
$weight reg (default), med, semi
$transform none (default), uppercase, lowercase

@mixin tn-weight($weight)

Mofify the font-weight independently.

Acceptable arguments
$weight light, reg (default), med, semi, bold

Function

tn-space($size, $modifiable)

Accepts either keyword or increments for size. The second argument indicates whether the value is modifiable based on the $tn-space-modify feature flag.

Example:

.foo {
    height: tn-space(20); //80px
    padding: tn-space(m); //40px
}

If $tn-space-modify is set to 'compact', the spacing value will be cut in half. Primarily enabled for collection components when information density is greater.

Example:

.bar {
    padding: tn-space(m,true); //20px
}
Acceptable arguments
$size base,xs,s,reg (default),m,l,xl
$modifiable true, false (default)
